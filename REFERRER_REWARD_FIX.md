# Исправление логики выдачи наград реферерам

## 🎯 Проблема

Система не выдавала награды реферерам после первого платежа их друзей (новых клиентов, использовавших их коды). Проблема возникла из-за того, что:

1. **Система была неработоспособна ~15 дней** - ничего не обрабатывалось
2. **После восстановления** были обработаны все застрявшие работы, но логика имела баг:
   - Если `first_payment_completed = TRUE`, функция возвращалась раньше времени
   - Не проверяла, была ли уже выдана награда рефереру
   - Это приводило к пропуску наград для клиентов, у которых флаг был установлен, но награда не была выдана

## ✅ Реализованные исправления

### 1. Исправлена логика `processPaymentCompletion`

**Файл:** `lib/webhooks/giftcard-processors.js`

**Изменения:**
- Вместо раннего возврата при `first_payment_completed = TRUE`, теперь проверяется, была ли уже выдана награда
- Добавлена проверка: сравнивается количество друзей, завершивших платеж, с `total_referrals` реферера
- Если награда не была выдана, она выдается даже если флаг уже установлен
- Предотвращает дублирование наград через проверку статистики

**Логика:**
```javascript
// Считаем друзей, завершивших платеж
const friendsCompleted = COUNT(friends WHERE first_payment_completed = TRUE)

// Сравниваем с фактическими рефералами реферера
const actualReferrals = referrer.total_referrals

// Если friendsCompleted > actualReferrals, награда не была выдана
if (friendsCompleted > actualReferrals) {
  // Выдать награду
}
```

### 2. Созданы скрипты для анализа и исправления

#### `scripts/find-missing-referrer-rewards.js`
- Находит всех клиентов, которые завершили первый платеж, но их реферер не получил награду
- Фильтрует самопереходы (self-referrals)
- Показывает статистику и детали по каждому случаю

#### `scripts/issue-missing-referrer-rewards.js`
- Выдает пропущенные награды реферерам
- Создает новую подарочную карту, если у реферера её нет
- Загружает $10 на существующую карту, если она есть
- Отправляет email уведомления
- Поддерживает режим `--dry-run` для тестирования

#### `scripts/check-specific-referral-cases.js`
- Проверяет конкретные случаи, упомянутые пользователем
- Проверяет баланс подарочных карт через Square API
- Показывает детальную информацию по каждому случаю

## 🔧 Как использовать скрипты

### 1. Найти пропущенные награды:
```bash
node scripts/find-missing-referrer-rewards.js
```

### 2. Проверить конкретные случаи:
```bash
node scripts/check-specific-referral-cases.js
```

### 3. Выдать пропущенные награды (dry-run):
```bash
node scripts/issue-missing-referrer-rewards.js --dry-run
```

### 4. Выдать пропущенные награды (реально):
```bash
node scripts/issue-missing-referrer-rewards.js
```

## ⚠️ Важные замечания

1. **Проверка баланса карт**: Скрипт проверки баланса требует правильной настройки Square API ключей
2. **Дублирование наград**: Исправленная логика предотвращает дублирование, но рекомендуется проверить перед массовой выдачей
3. **Временные метки**: Награды могли быть выданы в неправильное время (до реального платежа). Скрипты помогут найти и исправить такие случаи
