## ✅ Таблицы с данными (важные)

### 1. **square_existing_clients** - 7,265 записей
**Назначение:** Основная таблица клиентов из Square (legacy система)
- **Почему важна:** Содержит основную информацию о клиентах
- **Источник данных:** Webhooks от Square (customer.created, booking.created)
- **Статус:** Активно используется

### 2. **giftcard_jobs** - 4,921 записей
**Назначение:** Очередь задач обработки подарочных карт
- **Почему важна:** Управляет асинхронной обработкой webhook'ов
- **Источник данных:** `lib/workflows/giftcard-job-queue.js`
- **Статус:** Активно используется

### 3. **giftcard_runs** - 4,919 записей
**Назначение:** Отслеживание выполнения обработки подарочных карт
- **Почему важна:** Логирование всех этапов обработки
- **Источник данных:** `lib/runs/giftcard-run-tracker.js`
- **Статус:** Активно используется

### 4. **ref_clicks** - 431 записей
**Назначение:** Клики по реферальным ссылкам
- **Почему важна:** Отслеживание активности реферальной программы
- **Источник данных:** `app/api/track-referral-click/route.js`
- **Статус:** Активно используется

### 5. **processed_events** - 180 записей
**Назначение:** Идемпотентность обработки событий
- **Почему важна:** Предотвращает дублирование обработки
- **Источник данных:** `app/api/webhooks/square/referrals/route.js` (processPaymentCompletion, processBookingCreated)
- **Статус:** Активно используется

### 6. **notification_events** - 80 записей
**Назначение:** События отправки уведомлений (SMS/Email)
- **Почему важна:** Отслеживание доставки уведомлений
- **Источник данных:** Код отправки уведомлений
- **Статус:** Активно используется

### 7. **customers** - 23 записей
**Назначение:** Основная таблица клиентов (новая система)
- **Почему важна:** Современная структура данных клиентов
- **Источник данных:** Миграция из square_existing_clients или создание через API
- **Статус:** Используется, но данных мало (возможно, новая система)

### 8. **ref_links** - 23 записей
**Назначение:** Реферальные ссылки клиентов
- **Почему важна:** Хранит реферальные коды и URL
- **Источник данных:** `scripts/generate-referral-links-for-all-customers.js`
- **Статус:** Используется

### 9. **square_gift_card_gan_audit** - 68 записей
**Назначение:** Аудит GAN (Gift Account Number) подарочных карт
- **Почему важна:** Отслеживание подарочных карт
- **Источник данных:** Специальные проверки/аудит
- **Статус:** Используется

### 10. **ref_matches** - 4 записей
**Назначение:** Совпадения рефералов с бронированиями
- **Почему важна:** Связывает клики с реальными бронированиями
- **Источник данных:** `app/api/webhooks/square/referrals/route.js` (processBookingCreated)
- **Статус:** Используется

### 11. **ref_rewards** - 2 записей
**Назначение:** Награды за рефералов
- **Почему важна:** Хранит информацию о наградах
- **Источник данных:** Webhook обработка платежей
- **Статус:** Используется

### 12. **analytics_dead_letter** - 4 записей
**Назначение:** Необработанные аналитические события
- **Почему важна:** Отслеживание ошибок аналитики
- **Источник данных:** Ошибки при обработке аналитических событий
- **Статус:** Используется (наличие записей означает, что были ошибки)
